The dependency of the visualization tools are 

    1. the [classifier]_stat.[ProfileType].txt, which contains data for accuracy and timing;
    2. the [classifier]_stat.[ProfileType].txt, which contains detailed confusion matrix result for increasing sample size;
    3. utils.txt, which contains the name of different classes involved. 

There are three templates in this directory. Simply copy them and change the filename inside to see different data.
